### 오늘 한것

- 로그인 후 로그인했을 때와 로그아웃 했을 때 상세페이지를 구현하였다.
- localStorage에 로그인 유무, 유저정보, 토큰을 저장했다!!

```jsx
const isLogin = window.localStorage.getItem("isLogin"); // 로그인 유무
const loginUserInfo = window.localStorage.getItem("loginUserInfo"); // 유저정보
const accessToken = window.localStorage.getItem("accessToken"); // 토큰

useEffect(() => {
  window.localStorage.setItem("loginUserInfo", loginUserInfo);
}, [loginUserInfo]);

useEffect(() => {
  window.localStorage.setItem("accessToken", accessToken);
}, [accessToken]);

useEffect(() => {
  window.localStorage.setItem("isLogin", isLogin);
}, [isLogin]);
```

- 상세페이지 내에서 scrap 버튼을 클릭시 서버에 요청을 보내고 오류없이 버튼 상태도 변경되었다.

```jsx
useEffect(() => {
  axios.get(`/service/${ServiceId}`).then((res) => {
    setScrapNum(res.data.scrapNum);
  });
});
// 처음 상세보기 페이지로 들어온 후 scrap한 숫자 받아와서 상태에 저장

const addScrap = () => {
  axios
    .post(`/scrap/${ServiceId}`, null, {
      headers: { authorization: `Bearer ${JSON.parse(accessToken)}` },
    })
    .then(() => {
      setIsScrap(true);
    });
};
// scrap버튼을 눌렀을 때 post요청으로 서버데이터를 변경하고 scrap유무 상태를 변경

const deleteScrap = () => {
  axios
    .delete(`/scrap/${ServiceId}`, {
      headers: { authorization: `Bearer ${JSON.parse(accessToken)}` },
    })
    .then(() => {
      setIsScrap(false);
    });
};
// scrap버튼을 눌렀을 때 delete요청으로 서버데이터를 삭제하고 scrap유무 상태를 변경
```
