프로젝트를 하던 중에, 한 컴포넌트 안에서 axios 요청을 두번 보내야 하는 상황이 생겼다.

```jsx
useEffect(() => {
  axios
    .get(`/scrap/${ServiceId}`, {
      headers: { authorization: `Bearer ${state.accessToken}` },
    })
    .then((res) => {
      setIsScrap(isScrap.data.isScrap);
    });
});

useEffect(() => {
  axios
    .get(`/subscribe/${ServiceId}`, {
      headers: { authorization: `Bearer ${state.accessToken}` },
    })
    .then((res) => {
      setIsSub(isSub.data.isSubscribe);
    });
});
```

axios 요청을 두번 보내고 있었는데 코드도 길어지고 중복되는 코드도 많고 뭔가 비효율적인 느낌이 들었다..

한번에 보내는 방법은 없을까..고민하다가 axios를 한번에 여러개 보낼 수 있는 방법을 찾게되었다!!

```jsx
useEffect(() => {
  if (state.isLogin) {
    axios
      .all([
        axios.get(`/scrap/${ServiceId}`, {
          headers: { authorization: `Bearer ${state.accessToken}` },
        }),
        axios.get(`/subscribe/${ServiceId}`, {
          headers: { authorization: `Bearer ${state.accessToken}` },
        }),
      ])
      .then(
        axios.spread((isScrap, isSub) => {
          setIsScrap(isScrap.data.isScrap);
          setIsSub(isSub.data.isSubscribe);
        })
      )
      .catch((err) => console.log(err));
  }
});
```

axios.all을 사용하여 한번에 보내는 방법이 있었고 받아온 데이터를 spread를 사용하여 각 데이터를 나눌 수도 합칠 수도 있었다!!! 요청도 2번뿐이 아니라 여러개를 보낼 수 있다!!

앞으로 같은 조건으로 보내는 axios요청은 한번에 보낼 수 있는 axios.all을 사용해야겠다 ㅎㅎ
