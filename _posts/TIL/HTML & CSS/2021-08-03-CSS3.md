## CSS selector

CSS selector는 CSS 의 특정 요소들을 선택하여 적용할 수 있게 한다

- * selector - 모든 요소를 선택한다

```jsx
* {
  color: green;
}
```

- class selector - 주어진 값을 class 속성값으로 가진 HTML 요소를 찾아 선택한다

```jsx
// HTML
<div class="class1">username</div>
// CSS
.class {color: red}
//class 선택자는 .을 사용한다
```

- ID selector- 주어진 값을 id 속성값으로 가진 HTML 요소를 찾아 선택한다

```jsx
// HTML
<div id="id1">username</div>
// CSS
#id1 {color: red}
//id 선택자는 #을 사용한다
```

### class selector와 id selector

- class selector는 속성 값을 두개 이상 가질 수 있어서 한 태그 안에서도 여러종류의 스타일을 적용할 수 있다
- class selector는 글자색이나 글자 굵기 등 스타일을 지정할 때 많이 사용한다
- class selector는 한 문서 안에 여러요소에 적용 할 수 있다
- id selector는 한 문서 안에 유일한 요소에 적용 할 수 있다
- id selector는 문서 안에 요소의 배치 방법을 지정할 때 많이 사용한다
- id selector는 우선순위가 class selector보다 높다

| id                                | class                 |
| --------------------------------- | --------------------- |
| #으로 선택                        | .으로 선택            |
| 한 문서에 단 하나의 요소에만 적용 | 여러 요소에 적용 가능 |
| 특정 요소에 이름을 붙이는데 사용  | 스타일 분류에 사용    |

## CSS flexbox

- flexbox로 레이아웃을 구성할 수 있다
- 박스가 수직으로 분할되는 것이 기본값이다

### display: flex

- display : flex는 부모 요소에만 적용할 수 있다
- 자식 박스의 방향과 크기를 결정하는 레이아웃 구성방법이다
- 기본값으로, 자식 요소는 왼쪽부터 차례대로 배치된다

### flex-direction

- flex-direction 은 부모 박스요소애 적용할 수 있다
- 자식 박스에 특별한 속성을 주지 않아도, 부모 요소에 의해 자식요소가 영향을 받는다

### flex

- flex는 자식 요소에 적용할 수 있다

```jsx
flex: 0 1 auto;
// flex: <grow> <shrink> <basis>
```

- grow(늘린다) - 아이템들의 basis를 제외한 여백 부분을 grow에 지정된 숫자의 비율로 나누어 가진다
- shrink(줄인다) - grow의 반대이다 0으로 세팅하면 베이식스보다 작아지지않아 고정폭을 쉽게 만들수있다
- basis(기본크기) - 아이템의 기본크기

## CSS 단위

- px: 화면에 크기를 고정하는 단위
- % : 부모 요소의 영향을 받아서 %을 쓸 수 있다.

- em

  - 자기자신(넓이) font-size 영향을 받는다.
  - font-size는 상속영향을 받는다.
  - 상속에 영향을 받기때문에 유지보수가 힘들다.

- rem: 최상단의 부모의 font-size에 영향을 받는다.

- vw (veiw width) / vh (veiw heigth)
  - 화면에 보이는 전체 사이즈이다.
  - 100이 최대 사이즈(화면전체 사용한다)
  - 화면 축소하면 축소하는 만큼 유동적으로 바뀜
