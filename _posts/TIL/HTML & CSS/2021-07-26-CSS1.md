## CSS

- CSS는 HTML과 같은 마크업 언어가 표현되는 방법을 결정한다. HTML이 웹 페이지의 구조를 담당한다면,

- CSS는 구조의 외부와 내부를 꾸미는 역활을 담당한다.

### CSS의 박스모델

- 박스를 구성하는 요소

<img width="438" alt="cssimg2" src="https://user-images.githubusercontent.com/85768164/147356112-f943053e-2046-4f29-bff5-604df03a6459.png">

border (테두리): 엘리먼트에 테두리 너비 사이즈, 색상, 스타일을 지정

```jsx
p {
  border: 1px solid red;
}
//1px의 빨간색 실선을 추가
```

margin (바깥 여백): 엘리먼트와 엘리먼트 사이의 간격을 지정

```jsx
p {
  margin: 10px 20px 30px 40px
}
//값을 4개 넣으면 각각의 값은 상, 우, 하, 좌 로 시계뱡향

p {
  margin: 10px 20px;
}
//값을 2개만 넣으면 상하, 좌우 방향

p {
  margin: 10px;
}
//값을 1개만 넣으면 모든 방향

p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
}
//margin 위치를 특정해 값을 넣어줄 수 도 있다.
```

padding (안쪽 여백): 엘리먼트의 상, 하, 좌, 우 여백을 한꺼번에 지정

```jsx
p {
  padding: 10px 20px 30px 40px;
}
//방향은 margin 과 동일
```

박스 크기 측정 기준

```jsx
#container {
  width: 300px;
  padding: 10px;
  border: 2px solid red;
}

// 300px (content)
//+ 10px (padding-left)
//+ 10px (padding-right)
//+ 2px  (border-left)
//+ 2px  (border-right)
// 총 324px

* {
  box-sizing: border-box;
}
//위 값을 적용하면, 모든 박스에서 여백과 테두리를 포함한 크기로 계산됨
```

### block, inline-block, inline의 특징

|                              | block | inline-block         | inline               |
| ---------------------------- | ----- | -------------------- | -------------------- |
| 줄바꿈 여부                  | O     | X                    | X                    |
| 기본적으로 갖는 너비(width)  | 100%  | 글자가 차지하는 만큼 | 글자가 차지하는 만큼 |
| width, height 사용 가능 여부 | O     | O                    | X                    |
