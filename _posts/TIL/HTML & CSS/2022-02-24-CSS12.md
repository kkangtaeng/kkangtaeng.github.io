## nesting

- nesting은 Sass의 유용한 확장 기능으로 선언을 중첩하는 기능이다.
- CSS는 후손 셀렉터의 경우 부모요소를 기술해야 하지만 SASS는 간단히 작성할 수 있다.
- 하지만 너무 깊은 nesting은 가독성을 나쁘게 하고 셀렉터를 복잡하게 하기 때문에 주의해야 한다.

```jsx
// CSS
.nav {
  width: 80%;
  height: 20px;
}

.nav ul {
  list-style: none;
}

.nav li {
  color: red;
}

// SASS
.nav {
  width: 80%;
  height: 20px;
  ul {
    list-style: none;
  }
  li {
    color: red;
  }
}
```

- 부모 요소의 참조가 필요한 경우 `&`를 사용한다.

```jsx
// 예시
.nav {
  color: red;
  $hover {
    text-decoration: underline;
  }
}
```



## 조건문과 반복문

### if

- Javascript와 같이 if문을 사용 가능하다.

```jsx
if(condition, true, false)

// 예시
$type: red;

div {
  color: if($type == red, red, blue); // color: red;
}
```

### @if, @else

- else if 문도 사용이 가능하다.

```jsx
// 예시
$type: red;

div {
  @if $type == red {
    color: red;
  } @else if $type == blue {
    color: blue;
  } @else {
    color: black;
  }
}
```

### @for

- for문도 사용이 가능하다.

```jsx
// 예시
@for $i from 1 through 3 {
  .item-#{$i} { width: 2em * $i; }
}

// 결과
.item-1 {
  width: 2em;
}
.item-2 {
  width: 4em;
}
.item-3 {
  width: 6em;
}
```

### @while

- while문도 사용이 가능하다.

```jsx
// 예시
$i: 6;
@while $i > 0 {
  .item-#{$i} { width: 2em * $i; }
  $i: $i - 2;
}

// 결과
.item-6 {
  width: 12em;
}
.item-4 {
  width: 8em;
}
.item-2 {
  width: 4em;
}
```

## 주석

- CSS는 `/* */`만 지원하지만 SASS는 `/* */`와 `//`를 모두 사용할 수 있다.
