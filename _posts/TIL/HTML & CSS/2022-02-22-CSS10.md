## transition

- transition은 애니메이션이 일정 시간에 걸쳐 진행되도록 한다.
- transition은 가상 클래스 선택자 또는 Javascript의 부수적인 액션에 의해 발동된다. (자동으로 발동되지 않음)

### transition-duration

- transition이 끝날 때까지 걸리는 시간을 정한다.

```jsx
// 예시
transition-duration: time
// 기본값은 0이다.
```

### transition-delay

- transition이 시작하는 시간을 지연시킨다.

```jsx
// 예시
transition-delay: time
// 기본값은 0이다.
```

### transition-property

- transition이 적용될 속성을 정한다.

```jsx
transition-property: none | all | property

// 예시
transition-property: all
```

|이름|설명|
|---|---|
|none|모든 속성에 적용하지 않는다.|
|all|모든 속성에 적용한다.|
|property|속성을 정한다.(여러개의 속성을 지정할 경우 쉼표로 구분)|

### transition-timing-function

- transition의 진행 속도를 정한다.

```jsx
// 예시
transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out
```

|이름|설명|
|---|---|
|linear|처음부터 끝까지 일정한 속도로 진행|
|ease|천천히 시작되어, 그다음에는 빨리지고, 마지막에는 다시 느려진다.(기본값)|
|ease-in|천천히 시작된다.|
|ease-out|천천히 끝난다.|
|ease-in-out|천천히 시작되어 천천히 끝난다.|
