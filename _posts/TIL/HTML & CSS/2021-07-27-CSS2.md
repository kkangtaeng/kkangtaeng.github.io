오늘은 HTML, CSS, Javascript를 이용해서 계산기를 만들어 보았다.

<img width="354" alt="CSSimg3" src="https://user-images.githubusercontent.com/85768164/147356932-cb952443-457a-4b98-b10c-13a19daba31b.png">

먼저 기본틀이 주어졌다. 이것을 CSS와 자바스크립트를 이용하여 디자인과 계산기능이 가능하게 만들어보았다.

<img width="357" alt="img4" src="https://user-images.githubusercontent.com/85768164/147356944-4783ebf8-ef1f-4b76-b977-93a8de940bb7.png">

먼저 위 계산기를 CSS를 이용하여 크기와 색상을 조금 변경해보았다. 나는 검정색을 좋아하고 깔끔한것을 선호하기 때문에 최대한 깔끔하게 만들어봤다 ㅎㅎㅎ

```jsx
.calculator {
  background-color: black;
  width: 350px;
  height: 550px;
  border-radius: 30px;
  padding: 30px 10px 0px 10px;
  border : solid 10px gray;
  border-bottom: solid 10px whitesmoke;
  border-top: solid 10px whitesmoke;
}
// 위 속성들을 변경해보았다.
```

그리고 Javascript를 이용하여 첫번째 칸 두번째 칸 그리고 연산자 칸까지 입력되게 한 후 그 계산을 한 값을 마지막 칸에 넣고 마지막에 초기화 버튼을 누르면 전체 초기화가 되게 하는 문제였다.
