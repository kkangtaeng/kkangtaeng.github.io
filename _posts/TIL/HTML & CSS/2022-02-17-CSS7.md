## Shadow

- 텍스트나 요소에 그림자 효과를 부여하기 위해 사용한다.

### text-shadow

- 텍스트에 그림자 효과를 부여한다.

```jsx
선택자 { text-shadow: Horizontal-offset Vertical-offset Blur-Radius Shadow-Color; }

// 예시
div {
    text-shadow: 5px 5px 5px red;
}
```

|프로퍼티값|단위|설명|생략|
|---|---|---|---|
|Horizontal-offset|px|그림자를 텍스트의 오른쪽으로 지정값만큼 이동|X|
|Vertical-offset|px|그림자를 텍스트의 아래로 지정값만큼 이동|X|
|Blur-Radius|px|그림자의 흐림도를 지정|O|
|Shadow-Color|color|그림자의 색상을 지정|O|

### box-shadow

- 요소에 그림자 효과를 부여한다.

```jsx
선택자 { box-shadow: Inset Horizontal-offset Vertical-offset Blur-Radius Spread Shadow-Color; }

// 예시
div {
    box-shaodw: inset 5px 5px 5px 5px red;
}
```

|프로퍼티값|단위|설명|생략|
|---|---|---|---|
|Inset|inset|inset 키워드를 지정하면 그림자가 요소 안쪽에 위치|O|
|Horizontal-offset|px|그림자를 텍스트의 오른쪽으로 지정값만큼 이동|X|
|Vertical-offset|px|그림자를 텍스트의 아래로 지정값만큼 이동|X|
|Blur-Radius|px|그림자의 흐림도를 지정|O|
|Spread|px|그림자를 더 크게 확장|O|
|Shadow-Color|color|그림자의 색상을 지정|O|

## Gradient

- Gradient는 2가지 이상의 색상을 혼합하여 그라데이션 효과를 주는데 사용한다.

### linear-gradient

- linear-gradient는 선형 그라데이션 효과를 주는데 사용한다.

```jsx
// 예시
linear-gradient( direction, color1, color2, …, color3 )
```

### direction

|프로퍼티값|설명|
|---|---|
|to bottom|위에서 아래로 그라데이션을 만든다.(기본값)|
|to top|아래에서 위로 그라데이션을 만든다.|
|to left|오른쪽에서 왼쪽으로 그라데이션을 만든다.|
|to right|왼쪽에서 오른쪽으로 그라데이션을 만든다.|
|ndeg|n도의 방향으로 그라데이션을 만든다.|

```jsx
// 예시
linear-gradient( to right, yellow, red )
// 그라데이션을 왼쪽에서 오른쪽으로 노란색과 빨간색 순서로 만든다.
```

### color

- 색만 입력했을 때는 색을 균등하게 배분하여 그라데이션을 만들지만 색 뒤에 백분율 또는 픽셀 등으로 길이를 입력하면 그 길이까지 해당 색을 사용한다.

```jsx
// 예시
linear-gradient( to right, yellow 50%, red 60%, purple )
// 그라데이션을 왼쪽에서 50% 까지는 노란색, 50%초과 60%이하에는 빨간색, 나머지는 보라색으로 만든다.
```

## radial-gradient

- radial-gradient는 원형 그라데이션 효과를 주는데 사용한다.

```jsx
// 예시
radial-gradient( shape size at position, color1, color2, ..., color3 )
```

- size: 크기를 정한다.
- position: 중심의 위치를 정한다.

### shape

|프로퍼티값|설명|
|---|---|
|circle|원|
|ellipse|타원(기본값)|

```jsx
// 예시
radial-gradient( circle, yellow, red, brown );
```

### size

|프로퍼티값|설명|
|---|---|
|fathest-corner|가장 먼 모서리에 닿을 만큼의 크기로 설정(기본값)|
|fathest-side|가장 먼 면에 닿을 만큼의 크기로 설정|
|closest-corner|가장 가까운 모서리에 닿을 만큼의 크기로 설정|
|closest-side|가장 가까운 면에 닿을 만큼의 크기로 설정|

```jsx
// 예시
radial-gradient( fathest-side at 30% 20%, yellow, red, brown );
```

### position

```jsx
// 예시
radial-gradient( at 30% 20%, yellow, red, brown );
// 가로 30% 세로 20%의 위치에 중심의 위치를 정한다.
```

### color

```jsx
// 예시
radial-gradient( yellow, red, brown );
// 원 중심으로 부터 색상이 정해진다.
// 선형 그라데이션과 같이 색상 뒤에 길이를 입력할 수 있다.
```
