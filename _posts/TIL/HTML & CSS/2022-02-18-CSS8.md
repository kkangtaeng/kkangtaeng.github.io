## @keyframes

- 애니메이션이 어떤 모양에서 어떤 모양으로 변할지 정한다.

```jsx
@keyframes name {
  from { ... }
  to { ... }
  n% { ... }
}

// 예시
@keyframes big {
  from {
    width: 0px;
    height: 0px;
  }
  to {
    width: 150px;
    height: 150px;
  }
}
```

|이름|설명|
|---|---|
|name|애니메이션의 이름|
|from|시작할 때의 모양|
|to|끝날 때의 모양|
|n%|n%일 때의 모양|

## animation

### animation-duration

- 애니메이션의 진행 시간을 정한다.

```jsx
animation-duration: time | inital | inherit

// 예시
animation-duration: 3000ms;
// 3초에 걸쳐 애니메이션이 실행된다.
```

|이름|설명|
|---|---|
|time|시간을 정한다.(기본값은 0이며 단위는 s 또는 ms 를 사용한다.)|
|inital|기본값으로 설정한다.|
|inherit|부모 요소의 속성값을 상속 받는다.|

### animation-delay

- 애니메이션의 시작을 지연시킨다.

```jsx
animation-delay: time | initial | inherit
// 속성은 위와 같다.

// 예시
animation-delay: 3s
// 애니메이션의 시작이 3초 후에 시작한다.
```

### animation-direction

- 애니메이션의 진행 방향을 정한다.

```jsx
animation-direction: normal | reverse | alternate | alternate-reverse | initial | inherit

// 예시
animation-direction: normal
// 정해진 기본값으로 애니메이션을 진행한다.
```

|이름|설명|
|---|---|
|normal|정해진 순서로 진행(기본값)|
|reverse|반대 순서로 진행|
|alternate|정해진 순서로 진행했다가 반대 순서로 진행|
|alternate-reverse|반대 순서로 진행했다가 정해진 순서로 진행|
|inital|기본값으로 설정|
|inherit|부모 요소의 속성값을 상속|

### animation-fill-mode

- 애니메이션이 진행되지 않을 때의 모양을 정한다.

```jsx
// 예시
animation-fill-mode: none | forwards | backwards | both
```

|이름|설명|
|---|---|
|none|애니메이션이 끝난 후 상태를 설정하지 않는다.(기본값)|
|forwards|애니메이션이 끝난 후 그 지점에 그대로 있는다.|
|backwards|애니메이션이 끝난 후 시작점으로 돌아온다.|
|both|애니메이션의 앞 뒤 결과를 조합하여 설정한다.

### animation-timing-function

- 애니메이션의 진행 속도를 정한다.

```jsx
// 예시
animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out
```

|이름|설명|
|---|---|
|linear|처음부터 끝까지 일정한 속도로 진행|
|ease|천천히 시작되어, 그다음에는 빨리지고, 마지막에는 다시 느려진다.(기본값)|
|ease-in|천천히 시작된다.|
|ease-out|천천히 끝난다.|
|ease-in-out|천천히 시작되어 천천히 끝난다.|

### animation-play-state

- 애니메이션이 진행 상태를 정한다.

```jsx
// 예시
animation-play-state: running | paused
```

|이름|설명|
|---|---|
|running|애니메이션을 진행.(기본값)|
|paused|애니메이션을 멈춤|

### animation

- 모든 애니메이션 속성값을 한번에 지정한다.
- 값을 지정하지 않은 속성값은 기본값으로 적용된다.

```jsx
animation: name duration timing-function delay iteration-count direction fill-mode play-state

// 예시
animation: none 0 ease 0 1 normal none running
```
