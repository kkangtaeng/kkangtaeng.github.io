### SELECT

- SELECT 명령어는 데이터베이스로부터 데이터를 선택하여 불러온다.

### SELECT DISTINCT

- 고유한 값만을 데이터베이스로부터 불러온다.

### FROM

- 결과를 가져올 데이터베이스 테이블을 명시한다.

```jsx
SELECT 레코드명 FROM 테이블명

//예시
SELECT name FROM Users // User 테이블로 부터 name 열을 불러온다.

SELECT * FROM Users // User 테이블의 모든 정보를 불러온다.
```

### WHERE

- 필터역활을 하는 쿼리문으로, 선택적으로 사용한다.

### WHERE 절의 연산자

|연산자|설명|
|===|===|
|=|같다|
|>|초과|
|<|미만|
|>=|이상|
|<=|이하|
|<> 또는 !=|같지 않다|
|BETWEEN|특정 범위 사이|
|LIKE|특정 값과 비슷한 값을 필터|
|IN|리스트의 값들과 일치하는 데이터를 필터|
|IS NULL|값이 없는경우|
|IS NOT NULL|값이 없는 경우를 제외|

```jsx
SELECT 레코드명
FROM 테이블명
WHERE 레코드명 = '옵션값' // 동일한 데이터 찾기

WHERE 레코드명 > '옵션값' // 특정값보다 초과되는 데이터 찾기

WHERE 레코드명 <> '옵션값' // 동일하지 않은 데이터 찾기

WHERE 레코드명 BETWEEN '옵션값1' AND '옵션값2' // 범위 사이의 데이터 찾기

WHERE 레코드명 LIKE 's%' // s로 시작하는 데이터 찾기

WHERE 레코드명 IN ('옵션값1', '옵션값2') // 리스트의 값들과 일치하는 데이터 찾기

WHERE 레코드명 IS NULL // 값이 없는 경우
```

### ORDER BY

- 결과 집합을 오름차순 또는 내림차순으로 정렬한다. 기본적으로 오름차순으로 정렬하며, 내림차순으로 정렬하려면 `DESC` 키워드를 사용한다.

```jsx
SELECT 레코드명1, 레코드명2
FROM 테이블명
ORDER BY 레코드명1, 레코드명2 ASC|DESC
```

### INSERT INTO

- 테이블에 새로운 레코드를 삽입할 때 사용한다.

```jsx
INSERT INTO 테이블명
VALUES ('값1', '값2', ...)
```

### UPDATE SET

- 테이블에 레코드를 수정할 때 사용한다.

```jsx
UPDATE 테이블명
SET 레코드명 = '수정값'
WHERE '옵션값'
```

### DELETE

- 테이블에 레코드를 삭제할 때 사용한다.

```jsx
DELETE FROM 테이블명
WHERE '옵션값'
```

### INNER JOIN

- 두 테이블 간 일치하는 레코드를 불러온다.

![스크린샷](https://user-images.githubusercontent.com/85768164/151006569-1f72d753-5eae-4808-a372-cc332617267a.png)

```jsx
SELECT 레코드명
FROM 테이블명1
INNER JOIN 테이블명2
ON 테이블명1.레코드명 = 테이블명2.레코드명
```

### OUTER JOIN

- 두 테이블의 모든 레코드를 불러온다.

![스크린샷](https://user-images.githubusercontent.com/85768164/151007030-c99a63ea-ea16-4af2-8e10-c2bdd7683b3c.png)

```jsx
SELECT 레코드명
FROM 테이블명1
OUTER JOIN 테이블명2
ON 테이블명1.레코드명 = 테이블명2.레코드명
```

### LEFT JOIN

- 왼쪽 테이블의 모든 레코드와 오른쪽 레코드의 일치하는 레코드를 가져온다.

![스크린샷](https://user-images.githubusercontent.com/85768164/151007817-a7804ff4-0c25-4fea-a2c1-01151e6c7adf.png)

```jsx
SELECT 레코드명
FROM 테이블명1
LEFT JOIN 테이블명2
ON 테이블명1.레코드명 = 테이블명2.레코드명
```

### RIGHT JOIN

- 오른쪽 테이블의 모든 레코드와 왼쪽 레코드의 일치하는 레코드를 가져온다.

![스크린샷](https://user-images.githubusercontent.com/85768164/151008005-20c532fa-3ae3-4f2d-93f4-cd0bb72da214.png)

```jsx
SELECT 레코드명
FROM 테이블명1
RIGHT JOIN 테이블명2
ON 테이블명1.레코드명 = 테이블명2.레코드명
```
