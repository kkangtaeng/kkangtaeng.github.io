## SQL

- SQL(Structured Query Language)은 데이터베이스 용 플고그래밍 언어로, 데이터베이스에 Query를 보내 원하는 데이터를 가져오거나 삽입할 수 있다.
- 데이터가 구조화된 테이블을 사용하는 데이터베이스에서 활용할 수 있다. (구조화된 Query 언어이다.)

## 데이터베이스 명령어

### 데이터베이스 생성, 삭제
- 데이터베이스 생성
```jsx
CREATE DATABASE 이름;
```

- 데이터베이스 삭제
```jsx
DROP DATABASE 이름;
```

### 데이터베이스 사용
- 데이터베이스를 이용해 테이블을 만들거나 수정, 삭제 등의 작업을 하려면, 먼저 데이터베이스를 사용하는 명령어를 입력해야한다.
```jsx
USE 이름;
```

### 테이블 생성
- `USE`를 이용해 데이터베이스를 선택했다면, 테이블을 만들 수 있다. 테이블은 필드(열)와 함께 만들어야 한다.

|필드명|필드 타입|그 외 속성|
|---|---|---|
|id|숫자|Primary key이면서 자동 증가하도록 설정|
|name|문자열 (최대 255개의 문자)||
|email|문자열 (최대 255개의 문자)||

```jsx
CREATE TABLE test (
    id int PRIMARY KEY AUTO_INCREMENTm
    name varchar(255),
    email varchar(255)
);
```

### 테이블 삭제

- 테이블 삭제
```jsx
DROP TABLE 이름;
```

- 테이블 내 데이터 삭제
```jsx
TRUNCATE TABLE 이름;
// 위 명령어는 테이블은 삭제하지 않고 테이블 내 데이터만 삭제한다.
```

### 테이블 변경
- 테이블의 열을 추가, 삭제 또는 수정할 수 있다.

- 테이블 열 추가

```jsx
ALTER TABLE 테이블명
ADD 열이름 데이터타입

//예시
ALTER TABLE Users
ADD Email varchar(255);
```

- 테이블 열 삭제

```jsx
ALTER TABLE 테이블명
DROP COLUMN 열이름

//예시
ALTER TABLE Users
DROP COLUMN Email;
```

- 테이블 열 변경

```jsx
ALTER TABLE 테이블명
ALTER COLUMN 열이름 데이터타입

//예시
ALTER TABLE Users
MODIFY COLUMN Email varchar(255);
```

### 테이블 정보 확인

```jsx
DESCRIBE 이름;
```
