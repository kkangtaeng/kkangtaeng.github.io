## Tree traversal

- 특정 목적을 위해 트리의 모든 노드를 한번씩 방문하는 것을 트리 순회라고 한다.

### 트리 순회의 종류

- 전위 순회
  - 루트에서 시작해 왼쪽의 노드들을 순차적으로 방문하고, 왼쪽의 노드 탐색이 끝나면 오른쪽 노드를 방문한다.

![스크린샷](https://user-images.githubusercontent.com/85768164/153869741-168f3736-1d0c-4796-9d86-8f60b1891605.png)

A -> B -> D -> E -> C -> F -> G

- 중위 순회
  - 루트에서 시작해 왼쪽의 노드들을 순차적으로 방문하고, 왼쪽의 노드 방문이 끝나면 오른쪽 노드를 방문한다.

![스크린샷](https://user-images.githubusercontent.com/85768164/153869741-168f3736-1d0c-4796-9d86-8f60b1891605.png)

D -> B -> E -> A -> F -> C -> G

- 후위 순회
  - 제일 왼쪽 끝에 있는 노드부터 방문하기 시작해서 루트를 거치지 않고 오른쪽으로 이동해 방문한뒤, 제일 마지막에 루트를 방문한다.

![스크린샷](https://user-images.githubusercontent.com/85768164/153869741-168f3736-1d0c-4796-9d86-8f60b1891605.png)

D -> E -> B -> F -> G -> C -> A

## BFS / DFS

### BFS (Breadth-First Search)

- 가까운 정점부터 방문하여 더이상 방문할 정점이 없을 때, 그 다음 떨어져 있는 정점을 순서대로 방문한다.
- 너비를 우선적으로 탐색하는 방법으로 너비 우선 탐색 이라고 한다.
- 주로 두 정점 사이의 최단 경로를 찾을 때 사용한다.

![스크린샷](https://user-images.githubusercontent.com/85768164/153869741-168f3736-1d0c-4796-9d86-8f60b1891605.png)

A -> B -> C -> D -> E -> F -> G

### DFS (Depth-First Search)

- 하나의 경로를 끝까지 탐색한 후, 찾는 데이터가 아니면 다음 경로로 넘어가 방문한다.
- 깊이를 우선적으로 탐색하는 방법을 깊이 우선 탐색 이라고 한다.
- 주로 한 정점에서 시작해서 다음 경로로 넘어가기 전에 해당 경로를 완벽하게 탐색할 때 사용한다.

![스크린샷](https://user-images.githubusercontent.com/85768164/153869741-168f3736-1d0c-4796-9d86-8f60b1891605.png)

A -> B -> D -> E -> C -> F -> G
