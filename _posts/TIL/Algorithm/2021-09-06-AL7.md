### 풀이

slice 메소드를 이용해 뒤번호와 나머지 앞번호를 나누고 앞번호 길이만큼 *로 바꾼후 리턴한다.

```jsx
function solution(phone_number) {
  let back = phone_number.slice(-4);
  let front = phone_number.slice(0, -4);
  let star = "";
  for (let n = 0; n < front.length; n++) {
    star += "*";
  }
  return star + back;
}
```
