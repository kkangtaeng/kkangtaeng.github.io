### 풀이

입력받은 숫자를 문자열로 변환 후 반복문을 돌려 자릿수 합을 구한 뒤 해당 수로 나누어 떨어지는지 확인한다.

```jsx
function solution(x) {
  let word = String(x);
  let result = 0;
  for (let n = 0; n < word.length; n++) {
    result += Number(word[n]);
  }
  if (x % result === 0) {
    return true;
  } else {
    return false;
  }
}
```
