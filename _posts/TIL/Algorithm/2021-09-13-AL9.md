### 풀이

Math.min 과 filter 함수를 사용해 보았다.

```jsx
function solution(arr) {
  if (arr.length === 1) {
    return [-1];
  }
  let small = Math.min(...arr);
  return arr.filter((ele) => {
    return ele !== small;
  });
}
```
