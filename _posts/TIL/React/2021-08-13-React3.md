## Props

- 컴포넌트의 속성을 의미한다. props는 성별이나 이름처럼 변하지 않는 외부로부터 전달받은 값으로, 웹 어플리케이션에서 해당 컴포넌트가 가진 속성에 해당한다.

### Props의 특징

- props는 부모(상위) 컴포넌트로부터 전달받은 값이다.
- props는 어떤 타입의 값도 넣어 전달할 수 있도록 객체의 형태를 가진다.
- props는 함부로 변경될 수 없는 읽기 전용 객체이다.

### Props의 사용 방법

1. 하위 컴포넌트에 전달하고자 하는 값과 속성을 정의한다
2. props를 이용하여 정의된 값과 속성을 전달한다
3. 전달받은 props를 렌더링한다

```jsx
function Parent() { // Parent 컴포넌트 선언
    return (
        <div className="parent">
            <Child text={"I'm the second child"} />
   // <Child>I'm the second child</Child> 방법도 가능하다.
   // text라는 속성을 선언하고 이 속성에 위 문자열을 할당하여 Child 컴포넌트에 전달한다.
        </div>
    );
};

function Child(props) { // Child 컴포넌트 선언
    return (
        <div className="child"></div>
            <p>{props.text}</p> // I'm the second child
   // <p>{props.children}</p> 두번째 방법을 사용할 때는 children을 사용한다.
   // Parent 컴포넌트에서 전달한 문자열을 받아온다.
        </div>
    );
};
```

## State

- State는 컴포넌트 내부에서 변할 수 있는 값을 의미한다.

### useState의 사용법

- React에서는 state를 다루는 방법 중 하나로 useState라는 함수를 사용한다.
- useState는 state 라는 변수를 선언한 것과 같으며, 아무 이름으로 지어도 된다.
- state 변수는 React에 의해 함수가 끝나도 사라지지 않는다.

```jsx
import { useState } from "react";
// useState를 사용하기 위해 React로부터 import 키워드를 이용하여 useState를 불러온다

function CheckboxExample() {

// useState를 컴포넌트 안에서 호출한다
  const [isChecked, setIsChecked] = useState(false);

//위 코드를 풀어쓰면 아래 코드와 같다
    const useStateArray = sueState(false);
    const isChecked = useStateArray[0];
    const setIsChecked = useStateArray[1];

//useState 수도 코드
    const [state 저장 변수, state 갱신 함수] = useState(상태 초기값)
```
