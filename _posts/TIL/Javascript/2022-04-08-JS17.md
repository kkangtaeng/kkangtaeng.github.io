### 함수

Typescript의 함수 선언 방식은 기존 Javascript의 함수 선언 방식에서 인자와 함수의 반환 값에 타입을 추가해야 한다.

```jsx
// 예시
function sum(a: number, b: number): number {
    return a + b;
}

// 화살표 함수
let sum = (a: number, b: number): number => {
    return a + b;
}

// 화살표 함수를 이용해서 타입을 지정할 수 도 있다.
let sum: (a: number, b: number) => number {
    return a + b;
}
```

### 인자

Typescript는 함수의 인자를 필수 값으로 요구하며, `undefined`나 `null`이라도 인자로 넘겨야 한다. 즉, 정의된 인자값만 받을 수 있고 추가로 인자를 받을 수 없다.

```jsx
function sum(a: number, b: number): number {
  return a + b;
}
sum(1, 2); // 3
sum(1, 2, 3); // error
sum(1); // error
```

만약 인자를 필수로 넘기지 않아도 error가 안되게 하려면 `?`를 이용할 수 있다.

```jsx
function sum(a: number, b?: number): number {
  return a + b;
}
sum(1, 2); // 3
sum(1, 2, 3); // error
sum(1); // 1
```

Rest문법도 인자로 사용할 수 있다.

```jsx
function sum(a: number, ...nums: number[]): number {
  let total = 0;
  for (let num in nums) {
    total += nums[num];
  }
  return a + total;
}
sum(1, [1, 2, 3]); // 7
```
