## 동기(Synchronous)

- 요청에 대한 결과가 동시에 일어난다 즉, 요청을 보낸 후 해당 요청의 응답을 받아야 다음 동작이 살행되는 방식을 말한다.

### 동기(Synchronous)의 예시

```jsx
1. 손님 1이 아메리카노를 주문한다.
2. 접수를 받은 직원이 아메리카노를 만든다.
3. 직원이 손님 1에게 아메리카노를 전달한다.
4. 손님 2가 카페라떼를 주문한다.
5. 접수를 받은 직원이 카페라떼를 만든다.
6. 직원이 손님 2에게 카페라떼를 전달한다.
```

- 위 방식에는 손님2는 손님1이 아메리카노를 받을 때까지 주문을 하지 못하고 기다려야한다.

## 비동기(Asynschronous)

- 요청에 대한 결과가 동시에 일어나지 않는다 즉, 요청을 보낸 후 응답이 완료되지 않아도 다음 동작을 실행하는 방식을 말한다.

### 비동기(Asynschronous)의 예시

```jsx
1. 손님 1이 아메리카노를 주문한다.
  1. 접수를 받은 직원이 아메리카노를 만든다.
  2. 아메리카노가 완성되면 직원이 손님 1을 부른다.
  3. 아메리카노를 손님 1에게 전달한다.
2. 손님 2가 카페라떼를 주문한다.
  1. 접수를 받은 직원이 카페라떼를 만든다.
  2. 카페라떼가 완성되면 직원이 손님 2를 부른다.
  3. 카페라떼를 손님 2에게 전달한다.
```

- 위 방식에는 주문을 blocking 없이 전부 받고 전달은 비동기적으로 이뤄진다 즉, 주문은 계속 받으며 커피가 완성이 되면 손님에게 전달하는 방식으로 진행되어, 동기(Synchronous)보다 효율적인 작업이 가능하다.

## Callback

- Callback 함수란 다른 함수의 인자로 함수를 전달하고, 이벤트가 발생한 후 인자로 전달한 함수가 다시 호출되는 것을 말한다.

```jsx
const printString = (str, callback) => {
  setTimeout(() => {
    console.log(str);
    callback();
  }, Math.floor(Math.random() * 100) + 1);
};

const printAll = () => {
  printString("A", () => {
    printString("B", () => {
      printString("C", () => {});
    });
  });
};
printAll();
```

- 위처럼 랜덤하게 숫자가 나와도 callback 함수로 인해, ABC 차례대로 콘솔에 찍히는것을 볼 수 있다.

- 하지만 callback 함수는 순차적으로 이어져셔 많아지게 되면 코드를 읽는 가독성과 관리가 어려워진다. 이를 콜백지옥이라고 부른다. 이를 보완하기 위해 Promise 가 나오게 된다.

## Promise

- Promise는 Callback과 기능은 동일하나 더 간결하고 가독성이 좋아지게 한다.
- Promise는 resolve와 reject를 인자로 받는다.
  - resolve: 성공시 실행
  - reject: 실패시 실행

```jsx
const printString = (str) => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      console.log(str);
      resolve();
    }, Math.floor(Math.random() * 100) + 1);
  });
};

const printAll = () => {
  printString("A")
    .then(() => {
      return printString("B");
    })
    .then(() => {
      return printString("C");
    });
};
printAll();
```

- callback과 같은 값을 출력한다.

## Async/await

- 가장 최근에 나온 비동기 처리 패턴으로 Promise와 기능은 동일한 syntatic sugar이다.

```jsx
const printString = (str) => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      console.log(str);
      resolve();
    }, Math.floor(Math.random() * 100) + 1);
  });
};

const printAll = async () => {
  await printString("A");
  await printString("B");
  await printString("C");
};
printAll();
```

- promise와 같은 값을 출력한다.
