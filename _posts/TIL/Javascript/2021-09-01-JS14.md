## fetch

- fetch API는 네트워크를 통해 URL로 요청하는 것을 가능하게 해주는 API이다.

### fetch 사용법

- fetch는 인자로 URL을 받고, Promise 타입의 객체를 반환한다.

```jsx
let url = "https://test.com/testing";
fetch(url)
  .then((res) => res.json())
  // 자체적으로 있는 json()메소드를 사용해, 응답을 JSON형태로 변환시켜 다음 promise로 전달
  .then((json) => console.log(json))
  // 콘솔에 json을 출력
  .catch((err) => console.log(err));
  // 에러가 발생할 경우, 에러를 콘솔에 출력
```
