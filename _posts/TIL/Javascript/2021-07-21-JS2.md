## 조건문

- 조건문은 어떠한 조건을 판별하는 기준을 만드는 것이며 반드시 비교 연산자(comparison operator)가 필요하다.

### 비교연산자

| 연산자 | 설명   |
| ------ | ------ |
| >      | 초과   |
| <      | 미만   |
| >=     | 이상   |
| <=     | 이하   |
| ===    | 같다   |
| !==    | 다르다 |

조건문은 다음과 같이 쓸 수 있다.

```jsx
if (조건1) {
  // 조건1이 통과할 경우
} else if (조건2) {
  // 조건1이 통과하지 않고
  // 조건2가 통과할 경우
} else {
  // 모든 조건이 통과하지 않는 경우
}
//조건에는 불리언(Boolean)으로 결과가 나오는 비교구문이 들어간다.
if ((name = "tang")) {
  return true;
} else if ((name = "kkangtaeng")) {
  return true;
} else {
  return false;
}
// name 의 값이 'tang'이면 true값을 리턴
// name 의 값이 'tang'이 아니고 'kkangtaeng' 이면 true값을 리턴
// 둘 다 아니면 (name 의 값이 'tang'도 아니고 'kkangtaeng' 도 아니면) false 값을 리턴
```

두가지 조건이 한번에 적용되는 경우에는 논리 연산자(logical operator)를 사용한다.

### 논리연산자

| 연산자  | 설명                          |
| ------- | ----------------------------- |
| II(OR)  | 두 값 중 하나만 ture이면 ture |
| &&(AND) | 두 값 모두 true이면 true      |
| !(NOT)  | 반대값을 출력                 |

```jsx
if ((false, null, undefined, 0, NaN, ""))
  // false로 취급하는 falsy한 값 6가지
  // 나머지는 truthy한 값

  undefined || 10; // 10
5 || 10; // 5
//OR 연산자는 truthy한 값을 만나면, 그 값을 출력

undefined || false; // false
//둘 다 falsy한 값일 경우, 뒤에 있는 값을 출력

undefined && 10; // undefined
5 && false; // false
//AND 연산자는 falsy한 값을 만나면, 그 값을 출력

5 && 10; // 10
//둘 다 truthy한 값을 만나면, 뒤에 있는 값을 출력

//논리 연산자의 결과에는 불리언(Boolean)이 아닌값이 들어갈 수 있다.
```

### truthy, falsy

- if문에서 false로 변환되므로, if 구문이 실행되지 않는다

  if (false)

  if (null)

  if (undefined)

  if (0)

  if (NaN)

  if ('')

- 논리 연산자의 결과에 boolean 값이 아닌 값이 들어갈 수 있다

- or 연산자는 truthy한 값을 만나면, 그 값을 출력한다

```jsx
undefined || 10              // 10
5 || 10                      // 5
5 || console.log(실행되지 않음) // 5
```

- AND 연산자는 falsy한 값을 만나면, 그 값을 출력한다

```jsx
undefined && 10; // undefined
5 && false; // false
```

- 둘다 truthy한 값일 경우, 뒤에 있는 값을 출력한다

```jsx
5 && 10; // 10
```

- 둘다 falsy값일 경우, 뒤에 있는 값을 출력한다

```jsx
undefined || false; // false
```

## 문자열

문자열에 한 글자를 index로 접근하여 불러 올 수 있지만 해당 값을 변경할 수는 없다.

```jsx
var str = "KkangTaeng";
console.log(str[0]); // 'K'
condole.log(str[3]); // 'n'
console.log(str[20]); // undefined

str[0] = "A";
console.log(str); // 'AkangTaeng'이 되지는 않는다.
```

연산자를 쓸 수 있다.

```jsx
var str1 = "Kkang";
var str2 = "Tang";
var str3 = "1";
console.log(str1 + str2); // 'KkangTang'
console.log(str3 + 8); // '18'
//string 타입과 다른 타입 사이에 +연산자를 사용하면 string 타입으로 변환
```

문자열의 전체 길이를 반환할 수 있다.

```jsx
var str = "KkangTang";
console.log(str.length); // 9
```

## 메소드

| 메소드               | 설명                                                                                        |
| -------------------- | ------------------------------------------------------------------------------------------- |
| .indexOf(문자열)     | 문자열 내에서 특정한 문자열의 index값을 가져오고 찾고자 하는 문자열이 없으면 -1을 가져온다. |
| .lastIndexOf(문자열) | 문자열의 마지막에 등장하는 위치에 대한 index값을 가져온다.                                  |
| .includes(문자열)    | 배열이 특정 요소를 포함하고 있는지 불리언(Boolean)값으로 출력한다.                          |
| .split(문자열)       | 문자열을 인자로 주어진 문자열의 기준으로 쪼개서 배열로 반환한다.                            |
| .substring(시작, 끝) | 문자열에서 특정한 구간의 문자열을 반환한다.                                                 |
| .toLowerCase(문자열) | 대문자를 소문자로 변환한다.                                                                 |
| .toUpperCase(문자열) | 소문자를 대문자로 변환한다.                                                                 |
