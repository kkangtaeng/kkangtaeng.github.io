## 일급 객체

- 자바스크립트에서 함수는 특별한 대우를 받는 일급 객체이다.
- 자바스크립트에서 함수는
  - 변수에 할당할 수 있다.
  - 다른 함수의 인자로 전달될 수 있다.
  - 다른 함수의 결과로서 리턴될 수 있다.
- 함수를 변수에 할당할 수 있기 때문에, 함수를 배열의 요소나 객체의 속성값으로 저장할 수 있다.
- 이는 함수를 데이터(string, number, boolean, array, object) 를 다룰 수 있다는 것을 의미한다.

## 고차함수

- 고차 함수는 함수를 인자로 받을 수 있고, 함수의 형태로 리턴할 수 있는 함수이다.
- 콜백 함수는 다른 함수의 인자로 전달되는 함수이다.

### 다른 함수를 인자로 받는 경우

```jsx
function double(num) {
  return num * 2;
}

function doubleNum(func, num) {
  return func(num);
}
// doubleNum 함수는 다른 함수를 인자로 받는 고차 함수이다
// doubleNum 함수의 첫 번째 인자 func에 함수가 들어올 경우,
// func 함수는 doubleNum 함수의 콜백 함수이다

let output = doubleNum(double, 4);
console.log(output); // 8
// 위 같은 경우, double 함수는 doubleNum 함수의 콜백 함수이다
```

### 함수를 리턴하는 경우

```jsx
function adder(added) {
  return function (num) {
    return num + added;
  };
}
// adder 함수는 다른 함수를 리턴하는 고차 함수이다
// adder 함수는 인자 한개를 입력받아서 함수를 리턴한다
// 리턴되는 익명 함수는 인자 한개를 받아서 added와 더한값을 리턴한다

let output = adder(5)(3);
console.log(output); // 8
// adder(5)는 함수이므로 함수 호출 연산자 () 를 사용할 수 있다
```

### 함수를 인자로 받아서, 함수를 리턴하는 경우

```jsx
function double(num) {
  return num * 2;
}

function doubleAdder(added, func) {
  const doubled = func(added);
  return function (num) {
    return num + doubled;
  };
}
// doubleAdder 함수는 고차 함수이다
// doubleAdder 함수의 인자 func는 doubleAdder의 콜백 함수이다
// double 함수는 doubleAdder 함수의 콜백 함수이다

doubleAdder(5, double)(3); // 13
// doubleAdder(5, double)는 함수이므로 함수 호출 기호 () 를 사용할 수 있다
```

## 내장 고차 함수

### filter

- filter 함수는 모든 요소 중 원하는 값만 분류하여 반환한다.
- filter 함수 활용 예시
  - 배열의 각 요소가 특정 함수에 따르면, 사실(boolean)일 때, 따로 분류한다.
  - 🍎 🍓 🍰 .filter(과일) => 🍎 🍓 

```jsx
let arr = [1, 2, 3];
let result = arr.filter(function (el) {
  return el % 2 === 1;
});
result; // 1, 3

// arr 배열의 각 요소 중 홀수 인것만 분류하여 리턴한다
```

### map

- map 함수는 모든 요소에게 동일한 행동을 준 값에 대하여 모두 반환한다.
- map 함수 활용 예시
  - 배열의 각 요소가 특정 함수에 의해 다른요소로 지정된다.
  - 🥔 🥚 🌽 .map(요리) => 🍟 🍳 🍿

```jsx
let arr = [1, 2, 3];
let result = arr.map(function (el) {
  return el * 2;
});
result; // 2, 4, 6

// arr 배열의 각 요소에 2를 곱한것을 리턴한다
```

### reduce

- reduce 함수는 배열을 하나의 값으로 응축한다.
- reduce 함수 활용 예시
  - 배열의 각 요소를 특정 함수에 따라 원하는 하나의 형태로 응축한다.
  - 🥬 🥒 🫑 .reduce => 🥗

```jsx
let arr = [1, 2, 3];
let result = arr.reduce(function (acc, cur, idx) {
  acc += cur;
  return acc;
}, 1);
result; // 7

// arr 배열의 각 요소를 acc에 응축하여 리턴한다
```
