## JSON

- JSON은 서로 다른 프로그램 사이에서 데이터를 교환하기 위한 포맷이다. 객체는 타입변환을 이용해 String으로 변환할 경우 객체 내용을 포함하지 않아 [object object]라는 결과를 리턴한다. 따라서 객체를 JSON의 형태로 변환하거나 JSON을 객체의 형태로 변환하는 방법을 사용한다.

### JSON.stringify

- 객체 타입을 JSON으로 변환한다. 이같은 과정을 직렬화 한다고 한다.

```jsx
let message = {
  name: "김코딩",
  age: 20,
  message: "Hello world!",
};

let typeChange = JSON.stringify(message);
console.log(typeChange);
// `{"name": "김코딩", "age": "20", "message": "Hello World!"}`
```

### JSON.parse

- JSON을 객체 타입으로 변환한다. 이같은 과정을 역직렬화 한다고 한다.

```jsx
let packet = `{"name": "김코딩", "age": "20", "message": "Hello World!"}`;

let obj = JSON.parse(packet);
console.log(obj);
/*
let message = {
  name: "김코딩",
  age: 20,
  message: "Hello world!",
}
*/
```

### JSON의 규칙

- JSON은 자바스크립트의 객체와는 다른 작은 규칙이 있다.

||자바스크립트 객체|JSON|
|---|---|---|
|키|키는 따옴표('',"")없이 사용가능|반드시 큰따옴표("")를 사용|
|문자열 값|문자열 값은 어떠한 형태의 따옴표도 사용가능|반드시 큰따옴표("")로 감싸서 사용|
