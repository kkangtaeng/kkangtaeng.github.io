## 배열

- 배열은 순서가 있는 값이다
- 여기서 값은 element (요소) 라고 하고 순서는 index (인덱스)라고 부른다
- index 는 1이 아닌 0부터 번호를 매긴다

<img width="377" alt="_2021-06-28_19 48 14" src="https://user-images.githubusercontent.com/85768164/147482400-a1ad4fd4-2a2a-4b02-ab5b-88add92389f6.png">

```jsx
let myNumber = [73, 98, 86, 61, 96];

myNumber[3]; // 61
// myNumber라는 배열의 3번째 index를 조회한다

myNumber[3] = 200; // 200
// myNumber라는 배열의 3번째 index의 값을 변경한다

myNumber.length; // 4
//myNumber라는 배열의 길이를 알아낸다
```

### 배열이 빈배열인지 확인하기

```jsx
arr.length === 0;
```

### 배열의 반복

```jsx
for (let n = 0; n < myNum.length; n++) {
  console.log(myNum[n]);
}
// 배열의 element를 한번씩 출력하려면 위같은 반복문을 사용한다

let myNum = [10, 20, 40, 10];
let sum = 0;
for (let n = 0; n < myNum.length; n++) {
  sum = sum + myNum[n];
}
console.log(sum);
// 배열의 element를 다 더한 값을 출력하려면 위같은 반복문을 사용한다
```

## 배열의 메소드

### Array.isArray( ) - 특정값이 배열인지 판별하기

```jsx
Array.isArray(배열); // 배열인지 확인 후 boolean 값을 반환한다

let words = ["hello", "bye", "hi"];
Array.isArray(123); // false
Array.isArray("string"); // false
Array.isArray(words); // true
Array.isArray([1, 2, 3]); // true
```

### .push( ) - 마지막에 element를 추가한다

```jsx
let words = ["hello", "bye", "hi"];
words.push("good"); // 배열 마지막에 'good'을 추가한다
console.log(words);
// ['hello', 'bye', 'hi', 'good']
```

### .pop( ) - 마지막에 element를 제거한다

```jsx
let words = ["hello", "bye", "hi"];
words.pop(); // 배열 마지막에 'hi'를 제거한다
console.log(words);
// ['hello', 'bye',]
```

### .unshift( ) - 맨 앞에 element를 추가한다

```jsx
let words = ["hello", "bye", "hi"];
words.unshift("good"); // 배열 맨 앞에 'good'을 추가한다
console.log(words);
// ['good', 'hello', 'bye', 'hi']
```

### .shift( ) - 맨 앞에 element를 제거한다

```jsx
let words = ["hello", "bye", "hi"];
words.shift(); // 배열 맨 앞에 'hello'를 제거한다
console.log(words);
// ['bye', 'hi']
```

### .indexOf( ) - 해당 element의 index 값을 반환한다

```jsx
let words = ["hello", "bye", "hi"];
words.indexOf("hello"); // 0
words.indexOf("hi"); // 2
words.indexOf("good"); // -1 , 배열안에 해당 element가 나오면 -1을 반환한다
```

### .includes( ) - 해당 element의 유무를 boolean 값으로 반환한다

```jsx
let words = ["hello", "bye", "hi"];
words.includes("hello"); // true
words.includes("good"); // false
words.includes("Hello"); // false , 대소문자도 구분한다
```

### .concat( ) - 서로 다른 두 배열을 합친다

```jsx
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let arr3 = arr1.concat(arr2);
console.log(arr3);
// [1, 2, 3, 4, 5, 6]
```

### .slice( ) - 배열을 자른다

```jsx
let num = [1, 2, 3, 4, 5, 6];
console.log(num.slice(2)); // [3, 4, 5, 6] , 2번째 부터 자른다
console.log(num.slice(2, 4)); // [3, 4]       , 2번째 부터 3번째까지 자른다
console.log(num.slice(1, 5)); // [2, 3, 4, 5] , 1번째 부터 4번째까지 자른다
// slice(start, end) = start부터 end 미만까지 자른다

let num = [1, 2, 3, 4, 5, 6];
let sum = num.slice(1, 3);
console.log(num); // [1, 2, 3, 4, 5, 6]
console.log(sum); // [2, 3]
//배열 .slice() 는 원본 배열 손상 없이 똑같이 복제한다
```

### .splice( ) = 배열의 요소를 삭제 또는 교체 또는 추가한다

```jsx
let month = ["Jan", "March", "April", "June"];
months.splice(1, 0, "Feb"); // 1번째 인덱스에 삭제하지 않고 'Feb'를 추가한다
console.log(months); // ["Jan", "Feb", "March", "April", "June"]

months.splice(4, 1, "May"); // 4번째 인덱스에 1개를 삭제하고 'May'를 추가한다
console.log(months); // ["Jan", "Feb", "March", "April", "May"]
```

### find - 배열 중 첫번째 true값만 반환한다 만약 없으면 undefined 를 반환한다

```jsx
let arr = [1, 2, 3, 4, 5];

const result = arr.find((item) => {
  return item % 2 === 0;
});
console.log(result); // 2

//////

let userList = [
  { name: "Mike", age: 30 },
  { name: "Jane", age: 27 },
  { name: "Tom", age: 10 },
];

userList.find((user) => {
  if (user.age < 19) {
    return true;
  }
  return false;
});
console.log(result); // {name: "Tom", age: 10}
```

### arr.reverse( ) - 배열을 역순으로 재정렬한다

```jsx
let arr = [1, 2, 3, 4, 5];
arr.reverse(); // [5, 4, 3, 2, 1]
```

### arr.sort( ) - 배열을 재정렬한다

```jsx
let arr = [1, 5, 4, 2, 3];
arr.sort(); // [1, 2, 3, 4, 5]

//////

let arr = [27, 8, 5, 13];
arr.sort((a, b) => {
  return a - b;
});

console.log(arr); // [5, 8, 13, 27]
```

### forEach - 함수를 인수로 받아 배열을 반복한다

```jsx
let arr = ["Mike", "Tom", "Jane"];

arr.forEach((name, index) => {
  console.log(name); // Mike Tom Jane
  console.log(`${index + 1}. ${name}`); // 1.Mike 2.Tom 3. Jane
});
```

### join( ) - 배열을 합쳐서 문자열로 반환한다

```jsx
let arr = ["안녕", "나는", "철수야"];

let result = arr.join(); // 안녕,나는,철수야
let result = arr.join("-"); // 안녕-나는-철수야
let result = arr.join(" "); // 안녕 나는 철수야
```

### .filter() - 배열을 거른다

```jsx
let arr = [];
```

### Math.max - 가장 큰 숫자를 반환한다

```jsx
let arr = [1, 3, 2];
console.log(Math.max(...arr)); // 3

let arr = [-1, -3, -2];
console.log(Math.max(...arr)); // -1
```
