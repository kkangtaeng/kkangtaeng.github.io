## 스코프 (Scope)

- 스코프는 중괄호(블록) 또는 함수에 의해 나누어지는 유효 범위를 의미한다
- 바깥쪽 스코프에서 선언한 변수는 안쪽 스코프에서 사용이 가능하다
- 반대로, 안쪽에서 선언한 변수는 바깥쪽 스코프에서 사용이 불가능하다

```jsx
let username = 'Kkangtaeng';                     // 바깥쪽
if (username) {                                  // 안쪽
  let message = `Hello, ${username}!`;           // 안쪽
  console.log(message); // 'Hello, Kkangtaeng!'  // 안쪽
}                                                // 안쪽
console.log(message); // error                   // 바깥쪽
```

- 스코프는 중첩이 가능하다
- 가장 바깥쪽의 스코프는 전역(global) 스코프라고 부르고 그 안에 있는 스코프들을 지역(local) 스코프라고 부른다
- 지역 변수는 전역 변수보다 더 높은 우선순위를 가진다

```jsx
let name = 'taeng'; // 전역 변수
function userName() {
 let name = 'Kkang'; // 지역 변수
 console.log(name) // 두번째 출력 'Kkang' , 지역 변수에서 name을 다시 선언하여 그 값을 출력한다
}
console.log(name) // 첫번째 출력 'taeng' , 전역 변수 그대로 출력한다
userName
console.log(name) // 세번째 출력 'taeng'
```

### 스코프에서 변수를 찾는 팁

- 지역 변수와 전역 변수를 나눈다
- 선언된 변수를 찾고 지역 변수인지 전역 변수인지 명시한다
- 선언되지 않은 변수가 가리키는 대상을 찾는다 (이 때 안에서 밖으로 찾는다)

## var / let / const

||var|let|const|
|---|---|---|---|
|유효 범위|함수 스코프|블록 스코프 및 함수 스코프|블록 스코프 및 함수 스코프|
|재할당|가능|가능|불가능|
|재선언|가능|불가능|불가능|

## 클로저(closure)

- 클로저는 외부 함수의 변수에 접근할 수 있는 내부함수이다
- 클로저는 함수를 리턴하는 함수이다
- 리턴하는 함수에 의해 스코프가 구분된다
- 내부 함수는 외부 함수에 선언된 변수에 접근이 가능하다

```jsx
let adder = function(x) { // 외부 함수
	return function(y) { // 내부 함수
		return x + y; 
	}
}
let add5 = adder(1) // 외부 함수의 인자에 1을 넣는 add5를 선언한다
add5(5) // 6 , 내부 함수의 인자에 5를 넣는다
add5(7) // 8 , x + y (1 + 7) 을 리턴한다
```
