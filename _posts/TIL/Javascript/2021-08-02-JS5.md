# 객체

- 객체는 의미를 가진 값이다
- 객체는 키와 값 쌍으로 이루어져 있고 쉼표로 구분한다
- 객체는 중괄호를 이용해서 만든다

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};
// firstName = 키 , 'Taeng' = 값
// 키, 값 사이는 :(콜론)으로 구분한다
```

### 객체의 값을 사용하는 방법

- Dot notation - . 을 이용한 방법

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};
console.log(user.firstName); // 'Taneg'
console.log(user.city); // 'Incheon'
// user(객체).firstName(키)를 이용하여 'Taeng'(값)을 찾는다
```

- Bracket notation - [ ]를 이용한 방법

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};
console.log["firstName"]; // 'Taeng'
console.log["city"]; // 'Incheon'
// user(객체)['firstName(키)'] 를 이용하여 'Taeng'(값)을 찾는다
// Bracket notation을 사용할 때는 bracket 안쪽의 내용을 문자열 형식으로 전달해야 한다
```

### 객체의 값 추가, 삭제, 확인

- Dot notation과 Bracket notation 을 이용한 값 추가

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};

user.age = 28; // Dot notation을 이용하여 추가한다
user["isMale"] = true; // Bracket notation을 이용하여 추가한다

let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
  age: 28,
  isMale: true,
};
```

- delete 를 이용한 값 삭제

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};

delete user.lastName; // lastName의 키와 값 쌍을 삭제한다

let user = {
  firstName: "Taeng",
  city: "Incheon",
};
```

- in 을 이용하여 값 조회

```jsx
let user = {
  firstName: "Taeng",
  lastName: "Kkang",
  city: "Incheon",
};

"firstName" in user; // true
"age" in user; // false
```

## 메소드

### Object.keys( ) - 객체의 키를 배열로 반환한다

```jsx
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
console.log(Object.keys(obj)); // ['a', 'b', 'c']
```

### Object.values( ) - 객체의 값을 배열로 반환한다

```jsx
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
console.log(Object.keys(obj)); // ['1', '2', '3']
```

### Object.entries( ) = 객체의 키/값을 배열로 반환한다

```jsx
const user = {
  name: "Mike",
  age: 30,
  gender: "male",
};
Object.entries(user);
/*
[
	["name", "Mike"],
	["age", 30],
	["gender", "male"]
]
*/
```

### Object.fromEntries( ) - 배열의 키/값을 객체로 반환한다

```jsx
const arr =
[
	["name", "Mike"],
	["age", 30],
	["gender", "male"]
];
Object.fromEntries(arr);
/*
{
	name: 'Mike',
	age: 30,
	gender: 'male',
}
```

### Object.assign( ) - 객체를 복사한다

```jsx
const user = {
  name: "Mike",
};

const user2 = object.assign({}, user); // 여기서 {}은 초기값
user2.name = "Tom";

console.log(user); // name: "Mike"
console.log(user2); // name: "Tom"
```

배열과 객체를 반복해서 완벽하게 이해하기!!!
