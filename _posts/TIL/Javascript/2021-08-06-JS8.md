## DOM

- DOM은 Document Object Model의 약자로, HTML 요소를 javascript로 조작할 수 있다.
- DOM은 Javascript를 이용해서 엘리먼트의 속성값을 얻어내거나, 변경하는 방법이다.
- HTML의 구조가 Javascript 객체의 구조와 같이 트리구조로 되어 있어 DOM의 브라우저에 접근하기 가장 바람직하다.

### HTML에 Javascript 적용하기

- HTML에서 Javascript를 적용하기 위해서는 `<script>` 태그를 이용한다.

```jsx
<script src="index.js"></script>
```

### createElement() - 새로운 element를 만든다

```jsx
document.createElement("div"); // div element를 만든다

const Div = document.createElement("div"); // Div라는 변수에 div element를 할당한다
// 아직 아무것도 연결되있지 않기 때문에 변화가 없다
// 트리구조로 연결하기 위해 append를 사용한다
```

### append () - element에 넣는다

```jsx
docoment.body.append(Div); // 변수 Div에 담긴 div element를 body element에 append 한다
```

### querySelector / querySelctorAll - element를 조회한다

```jsx
const oneDiv = document.querySelector(".divclass");
// class명이 divclass인 element 중 첫번째 element를 조회한다

const Divs = document.querySelectorAll(".divclass");
// class명이 divclass인 모든 element를 유사 배열로 받아온다

const getOneTweet = document.getElementById("container");
const queryOneTweet = document.querySelector("#container");
console.log(getOneTweet === queryOneTweet); // true
// 위 코드는 querySelector와 비슷한 역활을 하는 오래된 방식으로 이해하면 된다
```

### textContent - element에 문자열을 입력한다

```jsx
const oneDiv = document.createElement("div");
oneDiv.textContent = "dev";
// Div 변수에 비어있는 div element에 'dev' 라는 문자열을 넣는다
```

### classList.add( ) / classList.remove( ) - class를 추가/제거 한다

```jsx
oneDiv.textContent = "dev";
oneDiv.classList.add(".divclass");
// class명이 divclass인 곳에 oneDiv element를 추가한다
// <div class="divclass">dev</div>

oneDiv.classList.remove(".divclass");
// class명이 divclass인 곳에 oneDiv element를 제거한다
```

### value

- DOM으로 `<input type="text">` 또는 `<textarea>` 와 같은 element의 입력값을 설정한다

```jsx
//example
console.log(document.querySelector("#username").value);

document.querySelector("#username").value = "새로운 값";
```
