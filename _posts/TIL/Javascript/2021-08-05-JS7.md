## Spread/Rest 문법

### Spread 문법

- 주로 배열을 풀어서 인자로 전달하거나, 배열을 풀어서 각각의 요소를 넣을 때에 사용한다.

```jsx
function sum(x, y, z) {
  return x + y + z;
}

const number = [1, 2, 3];

sum(...numbers); // 6

// ... 을 사용해 number에 있는 배열을 풀어서 인자로 전달되어
// 1 + 2 + 3 이 리턴되어 리턴값은 6이 된다.
```

### Rest 문법

- 인자를 배열의 형태로 받아서 사용할 수 있고, 인자의 개수가 가변적일 때 유용하다.

```jsx
function sum(...num) {
  return num.reduce((pre, cur) => {
    return pre + cur;
  });
}

sum(1, 2, 3); // 6
sum(1, 2, 3, 4); // 10

// 함수 정의에는 하나의 ... 만 존재할 수 있다.
// sum(...num1, ...num2) X

// 함수 정의에는 정의의 마지막 매개변수여야 한다.
// sum(...num1, num2) X
// sum(num1, ...num2) O
```

### 배열에서 사용하기

- 배열 합치기

```jsx
let arr1 = [0, 1, 2];
let arr2 = [3, 4, 5];
let arr3 = [...arr1, arr2];

return arr3; // [0, 1, 2, 3, 4, 5]

// ... 을 사용해 arr1의 배열과 arr2의 배열을 합친다
```

- 배열 복사

```jsx
let arr1 = [1, 2, 3];
let arr2 = [...arr1];
arr2.push(4);

return arr2; // [1, 2, 3, 4]

// ... 을 사용해 arr1을 arr2에 복사한다.(arr1.slice()와 유사)
// arr2에 4를 push 한다.
```

### 객체에서 사용하기

```jsx
let obj1 = { foo: "bar", x: 42 };
let obj2 = { foo: "baz", y: 13 };

let clonedObj = { ...obj1 };
let mergedObj = { ...obj1, ...obj2 };

return clonedObj; // { foo: 'bar', x: 42 };

return mergedObj; // { foo: 'baz', x: 42, y: 13 };

// 객체에서도 그대로 복사되고 중복되는 값은 나중에 입력되는 값으로 변경된다.
```

## 구조 분해

- Spread 문법을 사용하여 값을 해체한 후, 개별 값을 변수에 새로 할당하는 과정이다.

### 배열

```jsx
const [a, b, ...rest] = [10, 20, 30, 40, 50];

return a; // 10

return b; // 20

return rest; // [30, 40, 50]
```

### 객체

```jsx
const { a, b, ...rest } = { a: 10, b: 20, c: 30, d: 40 };

return a; // 10

return b; // 20

return rest; // {c: 30, d: 40}
```
