## 변수

### **변수 선언 및 할당**

- 변수는 이름이 붙은 값를 의미하고 상황에 따라 변할 수 있는 값이다.
- 변수 할당은 "같다"라는 의미가 아닌 "대입"이란 의미의 "="를 사용한다.

```jsx
let name; // 변수 선언
name = "Tang"; // 변수 할당
let name = "Tang"; // 동시에도 가능
```

- 표현식에서는 변수를 이렇게 사용할 수 도 있다.

```jsx
let sum = 1; // sum = 1
sum = sum + 2; // sum = 1+2
sum = sum + 3; // sum = 3+3
sum = sum + 4; // sum = 6+4
sum = 10;
```

## 함수

### 함수와 선언 방법

- 함수는 어떤 목적을 가진 작업들을 수행하는 코드들의 묶음이며, 입력값을 가질 수 있고 항상 출력 값을 반환한다.

반복적으로 실행이 필요한 경우 함수로 만들 수 있다!

```jsx
function sum(num1, num2) {
  // num1 과 num2 는 매개변수(parameter)
  let result;
  result = num1 + num2;
  return result;
}
console.log(sum(2, 4)); // 2 + 4 = 6
sum = 6;
```

### 함수 선언 방식

- 함수 선언식

```jsx
function sum(num1, num2) {
  let result = num1 + num2;
  return result;
}
```

- 함수 표현식

```jsx
const sum = function (num1, num2) {
  let result = num1 + num2;
  return result;
};
```

- 화살표 함수

```jsx
const sum = (num1 + num2) => {
	let result = num1 + num2;
	return result;
};
```

### 함수의 호출

- 변수에 함수가 담겨져 있다면, 다른 변수와는 다르게 () (소괄호)를 붙여서 함수를 호출할 수 있고 입력값을 설정할 수 있다

```jsx
let word = say("hello hi");
// 변수 word에 할당 된 함수 say에 문자열 'hello hi' 를 인자로 넣어 호출
// 'hello hi' 는 전달인자(argument), 줄여서 인자라고 한다
```

### 함수의 리턴

- 함수가 호출되고 나면, 함수의 {} (중괄호)에 있는 코드가 실행되고 결과값을 리턴(return)해야한다
- 아무것도 리턴(return) 하지 않으면 undefined를 리턴한다

## 타입

### 원시자료형

- Javascript에서 객체가 아니면서 method를 가지지 않는 6가지 타입
- 원시 자료형의 보관함인 변수에는 하나의 데이터만 담을 수 있다
- 원시 자료형이 담기는 보관함의 크기는 고정하는 것이 합당하다

```jsx
string, number, bight, boolean, undefined, symbol, null;
//원시 자료형의 6가지 타입
```

### 참조자료형

- 원시 자료형이 아닌 모든것
- 배열([]), 객체({}), 함수(function(){}) 가 대표적이다
- 하나의 주제는 있지만 분명 서로 다르고, 여러 개의 데이터를 가지고 있다

### typeof 연산자

- typeof는 변수의 데이터 타입을 반환하는 연산자이다

```jsx
typeof 변수; //사용 예시
```

- 반환되는 값

undefined : 변수가 정의되지 않거나 값이 없을 때

number : 데이터 타입이 수일 때

string : 데이터 타입이 문자열일 때

boolean : 데이터 타입이 불리언일 때 (ex. true or false)

object : 데이터 타입이 함수, 배열 등 객체일 때

function : 변수의 값이 함수일 때

symbol : 데이터 타입이 심볼일 때
