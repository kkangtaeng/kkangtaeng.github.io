## 웹 브라우저에 URL 입력하면 일어나는 일

![스크린샷](https://user-images.githubusercontent.com/85768164/155874302-bb3efb63-d20f-4a68-969c-df028e675fef.png)

1. 웹 브라우저 주소창에 웹 페이지의 URL 주소를 입력한다.
2. 입력한 URL 주소 중에서 도메인 이름을 DNS 서버에서 검색한다.
3. DNS 서버에서 해당 도메인 이름에 해당하는 IP 주소를 찾아 사용자가 입력한 URL 정보와 함꼐 전달한다.
4. URL 정보와 전달받은 IP 주소는 HTTP 프로토콜을 사용하여 HTTP 요청 메세지를 생성한다. 생성한 HTTP 요청 메세지는 TCP 프로토콜을 사용하여 해당 IP 주소의 컴퓨터로 전송된다.
5. 전달받은 HTTP 요청 메세지는 HTTP 프로토콜을 사용하여 웹 페이지 URL 정보로 변환된다.
6. 웹 서버는 도착한 웹 페이지 URL 정보에 해당하는 데이터를 검색한다.
7. 검색된 웹 페이지 데이터는 또 HTTP 프로토콜을 사용하여 HTTP 응답 메세지를 생성한다. 생성한 HTTP 응답 메세지는 TCP 프로토콜을 사용하여 전송된다.
8. 전달받은 HTTP 응답 메세지는 HTTP 프로토콜을 사용하여 웹 페이지 데이터로 변환된다.
9. 변환된 웹 페이지 데이터는 웹 브라우저에 의해 출력된다.
