## CLI 명령어

### pwd

- 현재 폴더가 위치한 경로를 확인하는 명령어

### mkdir + 폴더명

- 새로운 폴더를 생성하는 명령어

```jsx
mkdir <폴더이름>
// 명령어와 폴더의 이름을 space키로 구분
```

### ls

- 현재 폴더의 포함된 폴더나 파일을 확인하는 명령어
  |명령어|설명|
  |---|---|
  |ls -a|숨어있는 폴더나 파일을 포함한 모든 항목을 출력|
  |ls -l|폴더나 파일의 포맷을 전부 출력|
  |ls -la, ls -al|같이 사용도 가능|

### open.

- 현재 위치를 GUI로 실행하는 명령어

### cd + 폴더명

- 해당 폴더로 진입하는 명령어
  |명령어|설명|
  |---|---|
  |cd ~|홈 폴더로 진입|
  |cd /|최상위 폴더로 진입|
  |cd .|현재 폴더로 진입|
  |cd ..|상위 폴더로 진입|

### touch + 파일명

- 새로운 파일을 생성하는 명령어

### cat + 파일명

- 파일의 내용을 터미널에 출력하는 명령어

### rm + 폴더 or 파일명

- 폴더나 파일을 삭제하는 명령어

### mv + 폴더 or 파일명

- 폴더나 파일의 이름을 변경하거나 위치를 옮기는 명령어
  |명령어|설명|
  |---|---|
  |mv + 폴더 or 파일명 + 도착 폴더명|폴더나 파일을 이동|
  |mv + 폴더 or 파일명 + 변경 이름|폴더나 파일명을 변경|

### cp + 원본명 + 복사명

- 폴더나 파일을 복사하는 명령어

### whoami

- 현재 로그인된 사용자를 확인하는 명령어

### sudo

- 관리자 권한을 획득하는 명령어
